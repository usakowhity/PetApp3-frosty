# 🐾 **PetApp3 – Frosty Edition**  
**Portable Demo Version (Rabbit)**  

---

### 🇬🇧 **English Description**

The **Frosty Edition** is the official demo version of PetApp3.  
It includes pre‑generated photos and live‑action videos of **Frosty**, a frosty‑colored rabbit, so users can experience PetApp3 immediately without preparing their own materials.

This edition showcases **PetApp3’s real‑time emotional reactions**, using Frosty’s original photos and videos.

---

#### 📘 **What’s Included**
- Pre‑generated **photos and videos** for states **n1–p12**  
- Pre‑configured pet profile (species, appearance, affection word)  
- Ready‑to‑use PlayWindow setup  

---

#### 📘 **How to Use**
1. Download the demo ZIP from Releases.  
2. Extract it to any folder.  
3. Run **`run.bat`**.  
4. Frosty will appear and react in real time.  

---

#### 📘 **System Requirements**
- Windows 10 / Windows 11  
- No GPU required  
- Python included (portable)

### 🇯🇵 **日本語説明**

**PetApp3 – Frosty Edition（ポータブル版）** は、  
AIペットアプリ「PetApp3」の公式デモ版です。

白兎 **Frosty（Usako Whity）** の写真／実写ビデオがあらかじめ同梱されており、  
ユーザーは素材を準備せずにすぐに体験できます。

本エディションは、PetApp3 の **リアルタイム感情反応デモ** として位置づけられています。

---

#### 📘 **内容**
- 状態 **n1〜p12** の写真／ビデオを同梱  
- ペットプロフィール（種別・外見・魔法のことば）を設定済み  
- PlayWindow によるリアルタイム反応が可能  

---

#### 📘 **使い方**
1. ZIPファイルをダウンロードします。  
2. 任意のフォルダに展開します。  
3. **`run.bat`** を実行します。  
4. Frosty がリアルタイムで反応します。  

---

#### 📘 **動作環境**
- Windows 10 / Windows 11  
- GPU不要  
- Python同梱（ポータブル動作）
